Internot Reminders

This is a service that provides text message based single or recurring reminders. THIS SERVICE IS CURRENTLY IN BETA. For now, it is free, but I will probably need to charge something in order to cover the costs at some point.

To use this you have to send a text message to 1-888-880-4465 (this is a different number than the info service) with the right format depending on what you want.

Before you can add any reminders, you'll have to first set your timezone with the "location" command (see below)

Add a Reminder

To add a reminder, use the form remind me [future time] [optional: message in quotes]

For example:

remind me in 2 hours

remind me tuesday at 8 am "take out the trash"

remind me tuesday at 8 am to take out the trash will also work.

Add a Repeating Reminder

To add a reminder that repeats at a consistent interval, use the form every [interval]

For example:

every november 1st

every tuesday at 7pm "trash"

List Reminders

To list all reminders use: list

If you have too many reminders, we can't list them all for messaging cost reasons.

Remove Reminder

to delete a specific reminder, use: remove [id]

For example: remove 123

Your reminder id will be given to you when you create a reminder

Remove All Reminders

To delete all your reminders, use the command: remove all

Set Timezone

To set your timezone, use the form: location [location]

For example: location brooklyn, ny

Set 24 or 12 Hour Clock

To set your preferred time format, use the command Clock 12 or Clock 24

Provide Feedback

To provide feedback, use the form feedback [your feedback]

For example: feedback, an alert came too early!
